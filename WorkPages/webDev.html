Public Services: Web Application (2019 - 2020)

This project surged from the need to improve the current customer service system for the various public services in Ciudad Obregón, Sonora, Mexico. The project was inspired by the concept of smart cities, where technology is used to solve the new needs of society. The project development was supervised and designed by the teachers from Instituto Tecnológico Superior de Cajeme, who also provided the designs for the screens. As a base for the application design, the template AdmiLTE was used. For the development of the project technologies such as Angular and ASP.NET frameworks were used.
