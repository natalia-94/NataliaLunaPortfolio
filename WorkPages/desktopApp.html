Public Services: Web Application (2019 - 2020)

This is my first VR project that depicts a living room. The user can grab objects, interact with them, play sounds on objects and move across the room using the carpets on the floor (teleportation). This is a great environment for the user to get familiar with VR and the concepts of grab and move objects in VR. Made in Unity.
